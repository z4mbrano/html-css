Indrodução

Hoje veremos diversas tipos de mudança textual. Segue os exemplos embaixo 👇

NEGRITO

Negrito ou bould, na HTML4 usavasse o comando <b>, entretando na HTML 5, usamos o comado <strong>.

ITALICO

Italico, texto para aparecer mais ou citações, para isso usamos o comando <em>.

DELETADO

Sabe aquele rabisco? É esse, para isso use o comando <del>.

SUBLIME

O que te salvou muito no enem, sublinhar as palavras, para fazer isso na HTML, use o comando <ins>.

MARCATEXTO

É literalmente um marca texto, porém, para mudar sua cor precisa usar as css3, mas esse simples só precisa usar o comando <mark>.

CODIGO

Com o comando <code>, podemos mostrar come realmente é o codigo.

CITAÇÃO

Usado para citar falas de pessoas, personagens e etc.

TEXTO AO COTRÁRIO

Não entendeu? lê de trás pra frente 😉, daora né? Para fazer isso usamos o comando <bdo>

CITAÇÃO MAIOR

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto.
